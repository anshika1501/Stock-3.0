# **App Name**: StockCompass

## Core Features:

- Industry Portfolio Listing: Display a categorized list of stock portfolios (e.g., Banking, IT, Automotive), allowing users to browse by sector.
- Stock Index View: Present a comprehensive list of individual stocks within a selected industry portfolio, with essential overview details.
- Stock Detail Page: Show detailed information for a chosen stock, including company profile, key financial metrics, and current pricing data.
- Interactive Stock Chart: Render a basic, interactive line chart displaying historical price performance over configurable time periods for the selected stock.
- Stock Search & Filter: Enable users to efficiently search for specific stocks or apply filters within industry listings to narrow down results.
- AI-Powered Stock Insights: An AI tool that generates a concise summary or analyzes basic trends for a selected stock, offering quick, digestible insights.

## Style Guidelines:

- The primary color is a moderate, trustworthy blue (#2985CC), symbolizing reliability and data clarity.
- The background color is an extremely light, desaturated blue (#ECF4F9), providing a clean canvas that promotes readability and reduces eye strain.
- The accent color is a vibrant cyan (#2EDEEE), used sparingly to highlight interactive elements and call-to-actions, ensuring strong visual contrast.
- The primary typeface for all text (headlines, body, and data points) is 'Inter', a grotesque-style sans-serif known for its modern, objective, and highly readable design, ensuring clarity across various data displays.
- Utilize a set of minimalist, vector-based icons that clearly represent financial concepts, market trends, and navigational actions, maintaining a professional and intuitive interface.
- Employ a clean and structured layout with responsive grids for lists and focused single-column displays for detailed views, optimizing content presentation across devices and emphasizing data visualization over ornamentation.
- Implement subtle, functional animations for view transitions, data loading indicators, and chart interactions (e.g., tooltips), enhancing user experience without distracting from critical financial information.
